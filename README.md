# AuraHealth – AI Powered Healthcare Assistant 🏥🤖

AuraHealth is a modern, full-stack healthcare web application designed to act as a virtual assistant for patients and doctors. It leverages AI to provide symptom checking, health risk assessment, and multilingual support, all wrapped in a premium glassmorphic UI.

**[🔴 LIVE DEMO](https://aura-healthcare-assistent-demo.onrender.com/)**

![AuraHealth Banner](https://cdni.iconscout.com/illustration/premium/thumb/online-doctor-consultation-2675662-2234033.png)

## 🚀 Key Features

*   **🧠 AI Symptom Checker**: Chat with an intelligent bot to assess symptoms and get instant guidance.
*   **🗣️ Multilingual Voice Assistant**: innovative voice-to-text support for multiple languages (English, Hindi, Telugu, etc.).
*   **📊 Health Risk Dashboard**: Visualizes vitals (Heart Rate, SpO2) and calculates a real-time health risk score.
*   **🩺 Doctor & Patient Portals**: Dedicated dashboards for patients to view trends and doctors to manage appointments.
*   **🚨 Emergency SOS**: One-click distress signal that captures user location.
*   **💊 Medication Reminders**: Set and view daily pill schedules.
*   **📄 PDF Reports**: Generate comprehensive health summaries for medical consultations.

## 🛠️ Tech Stack

*   **Frontend**: HTML5, CSS3 (Glassmorphism), JavaScript (Vanilla), Chart.js
*   **Backend**: Python, Flask
*   **Database**: SQLite (SQLAlchemy ORM)
*   **AI/ML**: Custom Rule-indexed NLP (Multi-language), Heuristic Health Risk Analytics
*   **Deployment**: Gunicorn / Render / (Supports Heroku)

## 📦 Installation

To run this project locally:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/poojasrithalla06/AURA-HEALTHCARE.git
    cd AURA-HEALTHCARE
    ```

2.  **Install dependencies**
    ```bash
    python -m pip install -r requirements.txt
    ```

3.  **Run the application**
    ```bash
    python app.py
    ```

4.  **Access the App**
    Open `http://127.0.0.1:5000` in your browser.
